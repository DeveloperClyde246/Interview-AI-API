Requirement:
1. python 3.11 or below
2. pandas
3. matplotlib
4. tempfile
5. os
6. numpy
7. time
8. tensorflow
9. streamlit
10. cv2

How to Run:
1. open the folder in vscode
2. open new terminal and paste "streamlit run Deployment.py"
3. copy the link and paste in browser

Model Folder		: store all trained model
Function File   	: function to perform preprocessing
Deployment File		: output screen for facial expression and eye gaze detection
EDA File 		: EDA for dataset
Model File		: Model training and result file
